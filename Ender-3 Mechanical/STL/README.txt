# Summary

All aluminium extrusions and all mounting brackets converted into STL's to make a printable Ender 3! Feel free to use these files to help create mods and additional parts for an Ender 3!

You will still need to get hardware ie nuts and bolts. Official BOM can be found here:
https://github.com/Creality3DPrinting/Ender-3/blob/master/Ender-3%20Mechanical/Ender-3%20BOM.XLS

Whilst I wouldn't recommend printing the hotend plate and fan cover I have included them to complete the components needed.

Looking forward to seeing if anyone takes on the challenge to make one!
