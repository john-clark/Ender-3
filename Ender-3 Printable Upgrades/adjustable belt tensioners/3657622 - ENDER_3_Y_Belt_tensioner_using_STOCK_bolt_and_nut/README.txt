# Summary

I was looking for the Y belt tensioner using the stock nut and bolt for the for Ender 3 but I couldn't find anywhere. There might be one around but I just couldn't find it so I decided to make one myself.

# Print Settings

Printer Brand: Creality
Printer: Ender 3
Rafts: No
Supports: Yes
Resolution: 0.2
Infill: 20

Notes: 
Beside the "Y tensioner bearing", you can print everything in 0.2 layer height.

You need to print TWO "Y tensioner bearing". They're very thin (0.5mm) so you may want to print them in the finest layer height possible. I print them in 0.12 with ironing on my glass bed to get a smooth surface.
