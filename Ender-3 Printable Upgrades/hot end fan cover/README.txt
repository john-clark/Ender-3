# Summary

Ender-3 Hotend Cooler Fan Cover
It is a hot-end cooler cover for Ender-3.
It is a cover to prevent it because the life time of the cooler is shortened if the filament spider web generated from the nozzle is sucked into the cooler
(Google translation)

엔더-3용 핫엔드 쿨러 커버
엔더-3에 사용되는 핫엔드 쿨러 커버입니다.
필라멘트 거미줄이 쿨러내부에 빨려들어가면 쿨러의 수명이 단축되기 때문에 그것을 방지하기 위한 커버입니다.
